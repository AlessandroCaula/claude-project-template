---
name: plan
description: Save an implementation plan as a markdown file. Use when entering plan mode or when explicitly asked to create a plan, outline an approach, or design a solution.
---

When creating a plan, save it as a `.md` file in the current working directory. Name it `plan_<short_description>.md` based on the task (e.g., `plan_blast_pipeline.md`, `plan_annotation_extraction.md`).
